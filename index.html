Page Structure

Here is the code to get a basic page, like this one:


<!-- ////////// PAGE STRUCTURE DIV ////////// -->
<div data-role="page" id="page-structure">

	<div data-role="header">
		<h1>Page Structure</h1>
	</div><!-- /header -->

	<div data-role="content">	
		<p>Some text here.</p>
	</div><!-- /content -->

	<div data-role="footer">
		<h4><small>jQuery Mobile Overview - 2010 Andrew S. Roberts</small></h4>
	</div><!-- /header -->
</div><!-- /page -->

jQuery Mobile works right along with normal jQuery and loads each new page dynamically. The page structure might seem a bit odd to some, but all HTML content is contained in one single HTML page. Then with JavaScript, one page, or div, is shown, while the other div's are hidden. Dynamic transitions gives the effect that separate pages are being loaded.

Buttons & Icons

Numerous icons are available to use for buttons. Here are a few examples:


Delete Delete Delete Delete

Code:

<a href="index.html" data-role="button" data-icon="arrow-l">Delete</a>
		<a href="index.html" data-role="button" data-icon="arrow-r">Delete</a>
		<a href="index.html" data-role="button" data-icon="arrow-u">Delete</a>
		<a href="index.html" data-role="button" data-icon="arrow-d">Delete</a>

Buttons don't require text:

Delete

Code:

<a href="index.html" data-role="button" data-icon="delete" data-iconpos="notext">Delete</a>
